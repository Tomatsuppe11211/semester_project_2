# semester_project_2

## Description
This is an auction werbsite for learning purposes. This is NOT a real website with actual transactions!

## How to use
- Anyone can browse the listing items on the listing page
- Anyone can search for listings through key words in the descriptions or title
- Anyone can access the links in the footer
- Anyone can navigate to the register / login page (but not take actions without a valid noroff email)
- As a logged in user you can manage your own profile
- As a logged in user you can look at others' profiles
- As a logged in user you can create, edit and delete your own listings
- As a logged in user you can search for other users


## Technologies used
- HTML
- JS
- Tailwind CSS

## Available scripts
- npm run dev
- npm run build
